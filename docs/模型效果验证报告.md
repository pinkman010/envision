## 4. 局限性说明

### 4.1 已知限制

1. **格式依赖**：扫描版PDF识别准确率待验证
2. **多语言**：英文报告支持情况待测试，其他语言需人工复核
3. **复杂表格**：嵌套表格、跨页表格的解析准确率待验证

### 4.2 改进方向

1. 引入OCR增强模块提升扫描件识别率
2. 扩展多语言支持（日语、德语ESG报告）
3. 优化表格解析算法，支持更复杂的合并单元格

## 5. 测试方法说明

### 5.1 准确率计算方式

```
准确率 = 模型正确抽取的字段数 / 模型输出的总字段数
召回率 = 模型正确抽取的字段数 / 人工标注应抽取的字段总数
```

### 5.2 幻觉判定标准

- **显性幻觉**：抽取内容在原文中完全不存在
- **隐性幻觉**：抽取内容被原文支持，但存在事实性错误
- **相似度阈值**：Levenshtein相似度<0.98判定为疑似幻觉
