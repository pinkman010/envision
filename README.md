# ESG披露与沟通智能分析系统

## 项目简介

ESG披露与沟通智能分析系统是一套面向新能源企业的ESG智能分析工具，通过AI技术辅助企业提升ESG披露质量、识别核心实质性议题、优化ESG沟通策略。

## 核心功能

- **语料处理**: 支持PDF/Word文档解析、文本分块、向量化存储
- **议题检索**: 基于ISSB/SASB/HKEX标准识别核心实质性议题
- **差距分析**: 对照标准和同行报告进行差距分析
- **优化建议**: 根据分析结果生成针对性的披露优化建议
- **对标分析**: 多企业ESG指标横向对比
- **审计日志**: 全链路留痕，SHA-256哈希防篡改

## 技术架构

```
┌─────────────────────────────────────────────────────────┐
│                        UI层 (Streamlit)                  │
├─────────────────────────────────────────────────────────┤
│                        API层 (FastAPI)                   │
├─────────────────────────────────────────────────────────┤
│                      Agent层 (1+4)                       │
│  OrchestratorAgent（总控）                               │
│  ├── CorpusAgent      # 语料解析与向量化                  │
│  ├── RetrievalAgent   # RAG检索 + LLM议题识别            │
│  ├── AnalystAgent     # 差距分析（标准对照+同行对比）      │
│  └── AdvisorAgent     # 披露建议生成                      │
├─────────────────────────────────────────────────────────┤
│                      Utils层                             │
│  ChromaDB | LLM | 校验 | 相似度 | 审计 | 哈希 | 配置       │
├─────────────────────────────────────────────────────────┤
│                      数据层                              │
│              Chroma向量库 | SQLite数据库                  │
└─────────────────────────────────────────────────────────┘
```

## 快速开始

### 1. 克隆代码
```bash
git clone <repository_url>
cd esg-system
```

### 2. 配置环境变量
```bash
cp .env.example .env
# 编辑.env文件，填写LLM_API_KEY等配置
```

### 3. 安装依赖并运行
```bash
# 安装依赖
pip install -e .

# 启动后端API
uvicorn src.main:app --reload

# 启动前端UI（新终端）
streamlit run src/ui/app.py
```

### 4. 访问系统
- 前端UI: http://localhost:8501
- 后端API: http://localhost:8000

## 系统要求

- Python 3.10+
- Ollama (本地嵌入模型)
