# 用户操作手册

## 一、系统登录

### 1.1 访问系统
打开浏览器，访问系统地址：
- 本地部署: http://localhost:8501
- 服务器部署: http://<服务器IP>:8501

### 1.2 系统首页
首页展示系统概览信息：
- 已上传报告数量
- 已识别议题数量
- 系统状态

## 二、报告上传

### 2.1 上传ESG报告
1. 点击左侧导航栏"报告上传"
2. 点击"选择文件"按钮
3. 选择PDF或Word格式的ESG报告
4. 点击"上传并解析"按钮
5. 等待解析完成

### 2.2 查看解析结果
上传完成后，系统会显示：
- 文件名
- 文件大小
- 文本长度
- 分块数量
- 解析状态

## 三、议题识别

### 3.1 执行议题识别
1. 点击左侧导航栏"议题识别"
2. 选择已上传的报告
3. 点击"执行议题识别"按钮
4. 等待识别完成

### 3.2 查看识别结果
识别完成后，系统会显示：
- 识别到的实质性议题列表
- 每个议题的优先级
- 提取的量化数据
- 数据来源位置

## 四、对标分析

### 4.1 添加对标企业
1. 点击左侧导航栏"对标分析"
2. 点击"添加对标企业"按钮
3. 选择要对比的企业报告
4. 点击"确认添加"

### 4.2 查看对标结果
系统会展示：
- 各企业ESG指标对比
- 行业平均值
- 企业排名
- 优势与劣势分析

## 五、合规披露

### 5.1 执行合规检查
1. 点击左侧导航栏"合规披露"
2. 选择已上传的报告
3. 点击"执行合规检查"按钮

### 5.2 查看合规提示
系统会显示：
- 风险提示列表
- 对应的标准条款
- 改进建议
- 风险等级

## 六、审计日志

### 6.1 查看日志
1. 点击左侧导航栏"审计日志"
2. 选择查询条件：
   - 操作类型
   - 操作人
   - 时间范围
3. 点击"查询"按钮

### 6.2 导出日志
1. 设置查询条件
2. 点击"导出JSON"按钮
3. 保存日志文件

## 七、规则配置

### 7.1 修改议题规则
1. 点击左侧导航栏"规则配置"
2. 选择"议题规则"标签
3. 找到要修改的议题
4. 点击"编辑"按钮
5. 修改关键词或正则表达式
6. 点击"保存"按钮

### 7.2 备份规则
1. 点击"导出规则"按钮
2. 保存JSON文件

### 7.3 恢复规则
1. 点击"导入规则"按钮
2. 选择备份的JSON文件
3. 点击"确认导入"

## 八、人工复核

### 8.1 查看待复核内容
1. 点击左侧导航栏"人工复核"
2. 查看待复核的提取结果列表

### 8.2 执行复核
1. 点击"查看详情"按钮
2. 核对提取内容与原文
3. 如有错误，修改内容
4. 填写复核意见
5. 点击"确认复核"按钮

### 8.3 签名确认
复核完成后，系统会生成签名哈希，确保复核记录不可篡改。

## 九、常见问题

### Q: 上传文件失败？
A: 检查：
1. 文件格式是否为PDF或Word
2. 文件大小是否超过200MB限制
3. 文件是否损坏

### Q: 议题识别不准确？
A: 可以：
1. 在"规则配置"中修改议题规则
2. 添加更多关键词
3. 优化正则表达式

### Q: 如何导出分析报告？
A: 在各功能页面，点击"导出"按钮即可导出JSON格式的分析报告。

## 十、注意事项

1. **AI输出仅供参考**：所有AI生成的内容需经人工复核确认后方可使用
2. **定期备份数据**：建议定期备份./data目录
3. **保护API密钥**：不要将.env文件中的API密钥泄露给他人
4. **及时更新规则**：根据最新ESG标准更新规则配置
