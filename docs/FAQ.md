# 常见问题解答 (FAQ)

## 1. 系统使用相关

### Q1: 系统支持哪些文件格式？

**A**: 目前支持以下格式：
- PDF（推荐）：.pdf
- Word文档：.docx, .doc

单个文件大小限制：50MB

### Q2: 处理一份ESG报告需要多长时间？

**A**: 取决于报告页数和复杂度，以及网络环境和大模型API响应速度。具体时间需实测，预计：
- 50页以内：[待测试]
- 50-100页：[待测试]
- 100-200页：[待测试]

如超时请检查网络连接或联系管理员。

### Q3: 支持哪些披露标准？

**A**: 目前支持3套国际标准：
- **ISSB**: IFRS S1/S2 可持续披露准则
- **SASB**: 可持续会计准则委员会标准
- **HKEX**: 香港交易所ESG报告指引

国标（GB/T 36001等）正在开发中。

### Q4: AI抽取的结果可以信任吗？

**A**: 系统采用多重机制保证可靠性：
1. **字符级溯源**：点击任意结果可查看原文出处
2. **相似度校验**：自动比对AI输出与原文相似度
3. **置信度标记**：低置信度结果会提示人工复核

**重要**：所有对外披露内容仍需人工最终确认。

### Q5: 如何修改业务规则？

**A**: 业务规则存储在 `config/rule_templates/` 目录：
- `topic_rules.json`: 实质性议题定义
- `esg_standards.json`: 披露标准映射
- `match_rules.json`: 匹配逻辑规则

修改后重启服务生效。建议在测试环境验证后再应用到生产。

## 2. 技术部署相关

### Q6: 必须使用Docker部署吗？

**A**: 不是必须的，提供两种方式：
- **Docker部署（推荐）**：一键启动，环境隔离
- **源码部署**：适合开发调试，需要Python 3.10+

详见 `docs/部署说明文档.md`

### Q7: 可以使用国产大模型吗？

**A**: 可以。系统通过OpenAI兼容接口支持多种模型：
- 配置 `LLM_BASE_URL` 和 `LLM_API_KEY` 即可
- 已测试：文心一言、通义千问、智谱GLM等

注意：不同模型的效果可能有差异，建议先测试验证。

### Q8: 数据存储在哪里？安全吗？

**A**: 数据存储在本地：
- 结构化数据：`data/sqlite_db/`
- 向量数据：`data/chroma_db/`
- 原始语料：`data/raw_corpus/`

**安全措施**：
- 不上传任何数据到云端
- 审计日志带SHA-256哈希防篡改
- 支持定期备份

### Q9: 可以部署在内网环境吗？

**A**: 可以，但有以下要求：
- 内网需能访问大模型API（或部署本地模型）
- 需自行配置反向代理和HTTPS
- 建议配置防火墙规则限制访问IP

### Q10: 如何备份数据？

**A**: 备份以下目录：
```bash
data/sqlite_db/        # 数据库
data/chroma_db/        # 向量库
data/raw_corpus/       # 原始语料
data/export_results/   # 导出结果
```

建议设置定时自动备份任务。

### Q12: 为什么有些字段抽取为空？

**A**: 可能原因：
1. **原文确实未披露**：报告本身未提及该指标
2. **表述方式特殊**：使用了系统规则未覆盖的表达方式
3. **格式问题**：扫描版PDF、复杂表格可能影响识别

建议结合原文进行人工确认。

### Q13: 如何减少AI幻觉？

**A**: 系统已内置三重防护：
1. **温度=0**：强制确定性输出
2. **相似度校验**：低于阈值自动拦截
3. **JSON Schema**：强制结构化输出

如仍发现幻觉，请反馈至 `docs/DEV_LOG.md`

## 4. 业务相关

### Q14: 系统可以替代人工吗？

**A**: **不可以**。系统设计原则是「AI辅助、人工决策」：
- AI负责：信息提取、格式整理、风险提示
- 人工负责：专业判断、最终决策、对外披露

### Q15: 除新能源行业外，其他行业可用吗？

**A**: MVP阶段针对新能源行业优化，其他行业可用但效果可能下降。

扩展至其他行业需要：
1. 收集该行业ESG报告语料
2. 调整 `topic_rules.json` 中的议题定义
3. 重新验证效果

### Q16: 系统如何保证合规性？

**A**: 多重合规机制：
- **技术层面**：字符级溯源、审计留痕、哈希校验
- **流程层面**：人工复核、多级确认、不可篡改记录
- **法律层面**：明确声明AI输出仅供参考，企业承担最终责任
